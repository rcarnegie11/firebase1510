Minterms

The SOP canonical form is a logic sum of logic products where each logic product is known as a minterm. If the assignment is read as a binary number, associate an index with each row: 000 (0), 001(1), 010(2) ... 111 (7).

Sum of Products

Rules for generating sum of products form: a. Generate an AND term from the input variables for each row of the truh table where the function is true. b. If the input is false, complement it. c. Construct the Boolean connecting terms with the OR operator.
